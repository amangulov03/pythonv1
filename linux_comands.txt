ls - показывает все файлы и папки из той деректории в которой вы находитесь
ls -a - показывает файлы и директории а также скрытые файли и директории
ls -l - показывает более подробную информацию о файлах и директориях
ls -la - показывает дополнительную инф о файлах и папках + о скрытых файлах и папках тоже
ls путь до папки - показывает содежимое папки не заходя туда через путь 
ls Desktop/pythonV1/base - пример

cd - переход в домашнюю папку
cd ~ - переход в домашнюю папку
cd name - переходит в папку name, если такая папка существует в текущей деректории 
cd . - переходит в текущюю папку
cd .. - переходит на одну папку назад
cd ../.. - переходит на две папки назад

mkdir - создает директорию test в текущей папке
mkdir путь/test - создает директорию  test по пути
mkdir test1 test2 test3 - создст несколько папок: test1, test2, test3, в текущей папке

touch test.txt - создание файла test с расширением txt 

rm test - удаляет файлы
rmdir test - удаляет папку, если она пустая
rm -rf test - удаляет папку test, пустая она или не пустая не имеет значения

mv test1.txt test2.txt - переименовывает файлы и папки
mv test1.txt dirtest - переместит файл test1.txt в папку dirtest

history - показывет все команды, которые в данном терминале
history 10 - показывает последнию 10 команд в данном терминале

pwd - показывает путь до той директории где вы выполнили эту команду

nano test.txt - отроет файл test,txt в текстовом редакторе nano, если такого файла нет то создаст его 

cat test.txt - выведет в терминале содержимое файле test.txt

apt - системный установщик папкетов
snap - системный установщик папкетов (в основном для приложении)
sudo - super user do
